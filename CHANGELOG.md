# 更新日志

本文件记录用户可感知的主要变化，不代替提交历史。

## [Unreleased]

## [0.11.0] - 2026-07-22

### Added

- 新增单镜像自托管发布，Fastify 在一个进程中提供前端页面、`/api` 和 `/uploads`。
- 新增 GHCR 多架构发布流程，版本标签会构建 `linux/amd64` 与 `linux/arm64` 镜像，并执行匿名拉取和启动检查。
- 项目采用 MIT License，Copyright (c) 2026 lottshin。

### Changed

- Docker Compose 收敛为一个 `app` 服务，默认拉取固定的 `ghcr.io/lottshin/dingcard:0.11.0`，也保留单独的源码构建入口。
- 继续使用原有 `db` 和 `uploads` 命名卷，升级镜像不会自动删除 SQLite 数据库或上传文件。
- 从 `0.10.x` 升级时会先无损移除旧 `server/web` 容器；CI 会验证账号、草稿和图片可由新 `app` 继续读取。
- LocalStore 与 RemoteStore 仍是独立数据源，切换模式时不自动迁移账号、草稿或图片。
- 前端版本升至 `0.11.0`；Fastify 增加静态站点职责，服务端版本升至 `0.3.0`。

## [0.10.1] - 2026-07-20

### Added

- 新增 Markdown 卡片与自由画布工作区切换，两个工作区共用主题、账号和草稿入口。
- 自由画布支持嵌套场景、图层树、同级拖放排序、重命名、分组与取消分组。
- 图层支持锁定、隐藏、嵌套作用域编辑，以及与键盘和可访问树控件一致的选择行为。
- RemoteStore 支持草稿、托管图片上传、图片租约、回收和跨用户所有权保护。

### Changed

- 自由画布旧版 v1/v2 草稿会归一化迁移到 v3 嵌套场景模型。
- 嵌套内容的保存、重载、图片续租和导出统一遍历完整场景树，包括隐藏图层的资源引用。
- LocalStore 与 RemoteStore 保持独立数据源，切换 `VITE_API_BASE` 时不自动迁移账号、草稿或图片。
- 混合尺寸页面导出 ZIP 前会显示确认提示，图片保留各自页面尺寸。
- 前端版本为 `0.10.1`；服务端版本仍为 `0.2.0`。

### Fixed

- 修复嵌套组移动、缩放、旋转、对齐、吸附、复制粘贴时的世界坐标和历史记录问题。
- 修复数字属性输入在失焦、手势取消、草稿切换和负小数输入时提交旧值或丢失编辑缓冲的问题。
- 修复线条颜色与粗细控件在紧凑属性面板中的排版和十六进制颜色显示。
- 加固菜单、对话框、图层树、输入法组合和焦点恢复行为，避免隐藏工作区或旧异步请求误处理当前编辑状态。
